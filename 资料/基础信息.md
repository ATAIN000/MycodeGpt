## 开始前的准备
- **包下载**: ```npm install --registry=https://registry.npmmirror.com```
- **启动**: ```npm run dev```

# 项目概要

- **项目名称：** mycodegpt
- **项目简介：** 一个结合了大型语言模型的自动化代码项目
- **技术栈：**
  - **前端：** Vue3
  - **前端：** nodejs18
  - **前端：** ant-design-vue 4.2.1
  - **AI接口：** OpenAI GPT-4
